1、radio的checked无效

type=”radio”中checked属性无效。 “checked”属性的单选框会默认选中，但在vue里这是无效的，因为它会跟具体的参数值绑定。

解决方法：

vue官网教程：v-model 会忽略所有表单元素的 value、checked、selected 特性的初始值。因为它会选择 Vue 实例数据来作为具体的值。因此需要通过 JavaScript 在组件的 data 选项中声明初始值。：
