一：认识Hexo

Hexo 是一个快速、简洁且高效的博客框架，可以让你快速建立一个博客。

Hexo主题丰富，也适合对于编程无概念的人进行建站。

二：环境需求

安装博客：git + github账号 + Node.js + Coding账号 + Hexo

博客撰写：软件Typora（这是作者推荐，可用其他代替）

Ps：Coding部署网站需要一个成年人的身份证进行实名认证。

六：SSH配置（重要）

本地ssh建立

ctrl + c 结束本地浏览后，继续输入

再输入

将你的邮箱四字替换为你注册Coding和Github的邮箱，接着回车，

会出现询问你是否改路径的提示，回车（即什么都不输入）

接着是输入密码，回车（即什么都不输入）。

最后是确认密码，回车（即什么都不输入）。

到c盘，用户，你电脑的名字，,ssh文件夹，用记事本或其他软件打开id_rsa.pub文件，复制其中内容。

Coding ssh配置

打开Coding，登录后在右上角头像 -> 个人账户设置 -> 左侧 SSH公钥 -> 新增公钥 -> 内容栏内输入你刚才复制的SSH，勾选永久有效。

github ssh配置

打开github，鼠标点击右上角头像，setting，左侧找到SSH and GPG keys，右侧绿色按钮new SSH key，内容栏内输入你刚才复制的SSH。

九：部署网站

在博客根目录 右键 -> git bush here 输入

回到coding项目页面，选择持续部署中的网站托管

选择静态网站，网站名称任意，代码仓库对于你刚才建立的Coding仓库，网站类型选择静态！！而不是Hexo，节点香港节点

建立成功后选择部署，等待1分钟不到即出现网站地址，可以在互联网上访问到实例博客了。

十：总结和其他

一些感想

作者在Hexo博客建立上走了很多弯路，花费了约8个小时才摸索出方法，使用github建立时一直遇到样式丢失的错误，始终无法解决，而Coding的网上攻略相对较陈旧，希望可以帮助看的人建立自己的博客。
